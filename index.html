Cashflow

Finanzas personales premium — PWA instalable para el día a día

App completa para llevar tu dinero: cuentas y patrimonio, movimientos (gastos, ingresos, transferencias y recurrentes), presupuestos con alertas, objetivos de ahorro, inversiones e informes. Pensada para durar años, con una interfaz cuidada al detalle y pensada primero para el móvil. Uso personal y compartible en pareja, desplegada en producción.

instalable · offline · Web Push
  • Arquitectura hexagonal + DDD táctico + CQRS con outbox transaccional y 11 bounded contexts aislados (verificado con deptrac); el dominio no conoce Laravel.
  • Núcleo financiero: multi-cuenta, presupuestos con avisos por umbral, objetivos, cartera de inversiones e informes de gasto por categoría y tendencias.

  • Hogar compartido en pareja por invitación con enlace, y avisos Web Push automáticos (presupuestos y objetivos) disparados por eventos de dominio.

Atalaya

Panel de observabilidad y alertas para servicios en producción

Herramienta propia que vigila mis servicios desplegados: estado, métricas (CPU/RAM/red), logs, incidentes y gasto — todo en un panel único y con avisos al móvil por Web Push en cuanto algo se degrada, cae o se dispara el presupuesto. Proyecto de uso personal desplegado en producción.

Pull + Push
Doble modelo de ingesta
4
Bounded contexts
Web Push
VAPID · alertas al móvil
  • Arquitectura hexagonal + DDD táctico + CQRS: el dominio no conoce Laravel, que actúa solo como adaptador.
  • Doble ingesta: pull desde la GraphQL API de Railway y push agnóstico (cualquier servicio POSTea logs, métricas y latidos con un token).
  • Alertas por umbrales (CPU / RAM / latencia / error-rate y presupuesto) con push automático disparado por eventos de dominio.
  • Status page pública, vista de gasto del workspace y auto-diagnóstico (self-check) del propio panel.
